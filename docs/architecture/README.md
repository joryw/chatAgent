# 架构文档

本目录包含项目的架构设计、技术决策和实现细节文档。

## 📂 目录结构

### [概览 (Overview)](overview/)

系统整体架构和组件说明。

- 系统架构图
- 核心组件介绍
- 技术栈说明
- 数据流图

### [设计决策 (Design Decisions)](design-decisions/)

重要的技术决策记录 (ADR - Architecture Decision Records)。

- 技术选型
- 架构模式选择
- 设计权衡
- 决策背景和影响

### [实现细节 (Implementation)](implementation/)

具体的实现说明和技术细节。

- 模块实现
- 接口设计
- 数据模型
- 算法说明

## 🎯 目标受众

- **架构师**: 了解整体设计和决策
- **高级开发者**: 理解系统实现细节
- **技术负责人**: 评估技术方案

## 📖 推荐阅读顺序

1. [架构概览](overview/) - 先了解整体
2. [设计决策](design-decisions/) - 理解为什么这样设计
3. [实现细节](implementation/) - 深入技术实现
